No data is being stored as all you do is paste a script on official NEET site only to see your performance analysis. Your data stays completely safe and private.

Instructions

  1. Login to your NEET Portal
  2. Scroll down and click on "Click To View and Challenge OMR"
  3. Then click on "Click here to View and Challenge Recorded OMR" button
  4. Press "F12" or "Ctrl+Shift+I" on keyboard
  5. Find the "Console" Tab on top and go to it
  6. Paste the script code on Console Tab and press "Enter" (you will get the script code after selecting Test Booklet Code above)
  7. If you can't paste, type "allow pasting" & press "Enter" then paste the script code again
  8. Press "F12" or "Ctrl+Shift+I" on keyboard to close the developer tool
  9. Now you can view your performance analysis
